## 四位模2除法器

| 信号名 | 方向 | 描述      |
| ------ | ---- | --------- |
| X[3:0] | in   | 4位被除数 |
| Y[3:0] | in   | 4位除数   |
| R[2:0] | out  | 3位余数   |
| Q      | out  | 1位商     |

## CRC校验计算电路

| 信号名  | 方向 | 描述                  |
| ------- | ---- | --------------------- |
| A[7:0]  | in   | 8位原数据帧           |
| B[3:0]  | in   | 4位除数               |
| C[10:0] | out  | 8位原数据帧+3位校验码 |
